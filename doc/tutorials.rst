Tutorials
==========

These tutorials provide an overview about typical workflows to perform DFT+DMFT calculations with solid_dmft. The tutorials are sorted by complexity and introduce one after another more available features. The available tutorials are:


1. :ref:`csc_vasp_plo_cthyb </tutorials/SVO_os_qe/tutorial.ipynb>`: Typical one-shot DMFT calculation based on prepared hdf5 archive for SrVO3
2. :ref:`csc_vasp_plo_cthyb </tutorials/PrNiO3_csc_vasp_plo_cthyb/tutorial.ipynb>`:  full charge self-consistent (CSC) DFT+DMFT calculation using the PLO formalism with Vasp for PrNiO3
3. :ref:`csc_qe_w90_hubbardI </tutorials/Ce2O3_csc_w90/tutorial.ipynb>`: full CSC DFT+DMFT calculation using w90 in combination with Quantum Espresso utilizing the lighter HubbardI solver
4. :ref:`os_vasp_plo_cthyb </tutorials/NNO_os_plo_mag/tutorial.ipynb>`: one-shot magnetic DMFT calculation for NdNiO2 in a large energy window for 5 d orbitals
5. :ref:`plot_correlated_bands </tutorials/correlated_bandstructure/plot_correlated_bands.ipynb>`: plot the spectral function after a DFT+DMFT calculation
